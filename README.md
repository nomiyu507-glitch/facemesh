# Unmask Me: Fragments of Me

## Run

This project is a static front-end app built with `p5.js`, `ml5.js`, `HandPose`, and `FaceMesh`.

1. Start a local server in the project folder:

```bash
cd /
python3 -m http.server 8000
```

2. Open the app in your browser:

```text
http://localhost:8000
```

3. Allow camera access when the browser asks for permission.

## Notes

- Do not run it by opening `index.html` directly with `file://`.
- A local server is recommended so webcam access and external library loading work correctly.
